# Read-Only PowerShell Investigation Samples

These scripts are sample/reference content for AI Security Lab. They do not connect automatically to live services and do not modify users, sessions, devices, policies, files, or tenant settings.

Use them as portfolio examples or adapt them only in an approved lab/dev tenant after review.

Safety rules:

- No destructive actions.
- No tenant modification actions.
- No secrets, tokens, API keys, tenant IDs, or production data.
- Placeholder values only.
