# KQL Hunting Query Samples

These Microsoft Sentinel / Defender-style KQL examples are sample-safe references for AI Security Lab. Replace placeholders only in an approved lab or development workspace.

Use placeholders only:

- `user@example.com`
- `DEVICE-NAME`
- `203.0.113.10`
- `example.invalid`

Do not add real tenant IDs, real users, production IP addresses, or production URLs.
