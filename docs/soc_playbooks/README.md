# SOC Playbook Library

This folder contains sample-safe SOC investigation playbooks for AI Security Lab. They are written for local portfolio demos and Local SecOps Copilot retrieval.

All examples use fake/sample values only. Do not add real company data, tenant data, client data, vendor confidential data, production logs, secrets, tokens, passwords, API keys, real users, real domains, or tenant IDs.

## Playbooks

- Suspicious PowerShell
- Risky sign-in
- Phishing response
- Malware alert
- Impossible travel
- Mass file deletion
- Escalation matrix
